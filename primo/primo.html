PRIMO

PRIMO (Processing-in-Memory cOmpiter) was created to allow the adoption of PIM RVU without requiring any modifications to the existing programming environments, or special knowledge of the PIM architecture. At the same time, PRIMO automatically identifies the appropriate pieces of code to be executed in the PIM, and generates the relevant instructions. PRIMO adopts a hybrid code style, which means that host and PIM instructions will be mixed in the same code.

More details about PRIMO will be posted here soon. Meanwhile, it is possibe to check them in the Publications.
